# Tokonyadia

A Spring Boot learning project for building an e-commerce application.

## Description

Tokonyadia is a practice project designed to explore and learn Spring Boot fundamentals through building a simple online store application.

## Technologies Used

- Spring Boot
- Java
- Maven

## Branches

- `main` - Main branch
- `TanpaDTO` - Implementation without DTO pattern
- `DTO` - Implementation with DTO pattern
